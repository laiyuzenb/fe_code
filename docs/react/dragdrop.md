# 拖拽

## react-sortable-hoc 拖拽排序交换位置

### Features

- 高阶组件--与现有组件集成
- 拖动手柄、自动滚动、锁定轴、事件等！
- 超级流畅的动画--追逐 60FPS 的梦想 🌈
- 与虚拟化库协同工作：react-virtualized、react-tiny-virtual-list、react-infinite 等。
- 水平列表、垂直列表或网格 ↔ ↕ ⤡
- 支持触摸 👌
- 无障碍：支持键盘排序
